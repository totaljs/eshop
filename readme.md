## Installation

- install [__Node.js__ platform](https://www.nodejs.org)
- install [__GraphicsMagick__](http://www.graphicsmagick.org)
- `$ cd eshop`
- install `$ npm install total.js`
- install `$ npm install paypal-express-checkout`
